# AI-Powered Image Gallery, Search & Analysis
A Streamlit-based intelligent visual search system that allows users to:

- Upload images and store them in SQLite
- Generate image captions, emotion detection, and embeddings using Gemini 2.5 Flash
- Perform semantic visual search using cosine similarity
- View images in a gallery
- Use category filters
- Auto-generate Outfit Planner recommendations
- Ask questions in a Chat tab with gallery context awareness

# Features
- Image uploading & storage
- Gemini-powered captioning & analysis
- Visual similarity search
- Gallery with filters
- Outfit Planner & Chat integration

# Installation
pip install -r requirements.txt, 
streamlit run app.py
